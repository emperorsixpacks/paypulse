## Create checkout session

<ParamField body="plan_id" type="string" required>
  UUID of the plan to subscribe to.
</ParamField>

<ParamField body="success_url" type="string" required>
  URL to redirect after successful payment.
</ParamField>

<ParamField body="cancel_url" type="string" required>
  URL to redirect if payment is cancelled.
</ParamField>

<ParamField body="customer_email" type="string">
  Pre-fill the customer's email.
</ParamField>

<ParamField body="customer_name" type="string">
  Pre-fill the customer's name.
</ParamField>

<ParamField body="metadata" type="object">
  Arbitrary data passed through to webhooks.
</ParamField>

### Checkout statuses

| Status | Description |
|--------|-------------|
| `PENDING` | Awaiting customer payment |
| `COMPLETED` | Payment successful |
| `EXPIRED` | Session expired (24h TTL) |
| `CANCELLED` | Cancelled by merchant |
