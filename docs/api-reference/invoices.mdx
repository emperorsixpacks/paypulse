openapi: "/openapi.json" "GET /api/v1/invoices" "GET /api/v1/invoices/{invoice_id}"
---

<ParamField header="X-Api-Key" type="string" required>
  Your API key.
</ParamField>

### Invoice statuses

| Status | Description |
|--------|-------------|
| `PENDING` | Awaiting payment |
| `PAID` | Payment received |
| `FAILED` | Payment attempt failed |
| `VOID` | Invoice cancelled |

### Refund statuses

| Status | Description |
|--------|-------------|
| `NONE` | No refund applicable |
| `PENDING` | Refund initiated, awaiting processing |
| `PROCESSED` | Refund completed |
| `FAILED` | Refund attempt failed |
