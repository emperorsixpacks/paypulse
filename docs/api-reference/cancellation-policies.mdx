Cancellation policies define how refunds are calculated when a customer cancels a **FIXED** billing plan. METERED plans do not support refunds.

### Refund types

| Type | Behavior |
|------|----------|
| `none` | No refund on cancellation |
| `full` | Full refund if cancelled within `refund_window_days` |
| `percentage` | Fixed percentage of the plan amount refunded |
| `prorate` | Refund proportional to remaining time in the period |
