</Warning>

### Event types

| Event | Description |
|-------|-------------|
| `invoice.paid` | Invoice payment successful |
| `invoice.failed` | Invoice payment failed |
| `invoice.created` | New invoice generated |
| `subscription.created` | New subscription started |
| `subscription.active` | Subscription became active |
| `subscription.cancelled` | Subscription cancelled |
| `subscription.past_due` | Payment failed, past due |
| `subscription.expired` | Subscription expired after dunning |
| `checkout.completed` | Checkout session completed |
| `checkout.expired` | Checkout session expired |

### Verifying signatures

Every delivery includes an `X-Signature` header with an HMAC-SHA256 signature:

```python
import hmac, hashlib

def verify_webhook(payload: bytes, signature: str, secret: str) -> bool:
    expected = hmac.new(secret.encode(), payload, hashlib.sha256).hexdigest()
    return hmac.compare_digest(expected, signature)
```
